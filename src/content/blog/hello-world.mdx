This is the first post on my new blog. I built it to have a quiet, fast place to
write up the things I figure out — mostly about software, trading systems, and side
projects.

## Why bother with a blog in 2026?

Writing forces clarity. If I can't explain how something works in a few paragraphs,
I probably don't understand it yet. A few things I plan to write about:

- post-mortems of bugs that taught me something
- how my side projects actually work under the hood
- short notes on tools I like

## A taste of code

Posts support syntax-highlighted code blocks out of the box:

```python
def fib(n: int) -> int:
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

print([fib(i) for i in range(10)])
```
